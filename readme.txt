Eternal Lands User Menu files
------------------------------
User Menu infos || https://www.el-wiki.net/User_menus

Name		|Filename		|Description
-------------------------------------------------------------------------------------------------------------------
Usermenu	|1_usermenu.menu	|Emotions/Privacy/Beam me to Isla Prima/Ignore/Friend/View Only Storage
PMmenu		|2_pm.menu_off		|Direct PMs/Direct Guild Message/Direct Channel Message (edit this file before use!)
Guild		|3_guild.menu		|Guild User Commands
Guildmaster	|3_guildmaster.menu	|Guild Master Commands
URLs		|4_url.menu		|Useful URLs
Info		|5_info.menu		|Various information (server/guilds/users)
Misc		|6_misc.menu		|Command help/AFK/Save/Change Password/Exit
Chat Alerts	|sound_warnings.txt	|[PM from/#GM/You stopped harvesting./Day ends

Since the game loads the user menu files in alphabetical order, I put numbers in front of the filename.

Delete or rename the .menu files* in your Eternal Lands directory (e.g.: .menu_old)
*By default only info.menu and misc.menu are present.

Edit the file 2_pm.menu_off with your own Friends and Channels and rename the filename from 2_pm.menu_off to 2_pm.menu

Copy the .menu files in your Eternal Lands directory
